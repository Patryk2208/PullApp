# Long Distance Ride Planner

A simple web-based route planner using OpenStreetMap.
Built using React + TypeScript + Vite.

## How to Run Locally

### 1. Prerequisites
- Install [Node.js](https://nodejs.org)

### 2. Install Dependencies
Open a terminal in the project folder and run:

```bash
npm install

### 3. Run

```bash
npm run dev
